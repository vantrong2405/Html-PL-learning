Lưu ý : Thẻ thì có thẻ đóng và thẻ mở
h* : *-> số
a : thẻ link
b : thẻ in đậm
p : Thẻ văn bản
div : tHẻ chia vùng (mỗi vùng thì có 1 dòng )
br : Thẻ xuống dòng ký tự
input : Thẻ nhập (Có nhiều dạng)
form : Tạo form
img : Thẻ hình ảnh Đơn vị : gồm có nhiều loại nhưng chủ yếu chúng ta sài % và px ctr + / : Để đóng văn bản trên html
hr : là thẻ dấu gạch ngang
table : là thẻ bảng
trong table chưa thead và tbody
trong thead và tbody chứa tr (chứa dòng bảng)
trong tr lại có th(là dòng in đậm) và td(dòng thường)
